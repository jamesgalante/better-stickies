Why

Stickies, but this decade

macOS Stickies hasn’t changed since the ’90s. Better Stickies rebuilds the idea around macOS 26’s Liquid Glass: every note is a real glass pane that refracts and saturates whatever sits behind it. A note is just a close dot, a pin, and your words — everything else lives in the menu bar.

Notes live in one plain JSON file that scripts and agents can read and write while the app runs — changes merge in live.

Under the hood

How the glass works

There’s no public API for most of what these notes do. Each pane is an NSGlassEffectView in its clear style over a rewritten NSVisualEffectView backdrop — fog washes hidden, saturation boosted, sampling forced to full resolution — plus a handful of documented private-API overrides that keep the glass lively when windows lose focus.

Every override states the symptom it fixes and fails safe: if a future macOS renames the internals, notes fade to standard materials instead of breaking. The full story is in the source.

These cards use the same recipe the app discovered: backdrop-filter: blur(22px) saturate(1.6) — the web’s version of the note backdrop. What you’re reading is sitting on the formula.

FAQ

Questions

Is it safe to open an unsigned app?

The code is fully open — read it, build it yourself with Scripts/make_app.sh, and the release zip is produced by that same script. “Unsigned” only means there’s no $99/year Apple Developer membership behind it.

Where do my notes live?

In one JSON file: ~/Library/Application Support/BetterStickies/notes.json. Plain, documented, and safe for external tools to edit while the app runs — the app watches the file and merges outside changes without clobbering what you’re typing.

Does it run on older macOS?

It launches on macOS 14+, but the signature look needs the macOS 26 Liquid Glass engine. Older systems render a classic frosted blur instead.

Why doesn’t the glass dim when a note loses focus?

Deliberately. macOS dims glass on unfocused windows; sticky notes live unfocused, so the app keeps every pane lively via a few careful private-API overrides that fail safe on future macOS versions.

What does it cost?

Nothing. MIT licensed, no account, no telemetry — it’s a desktop app that writes a JSON file.
